La idea de este proyecto la tome de un curso de html que vi hace un tiempo y nunca pude desarrollar.

Con el curso de BA pude llevar a cabo la idea y trate de utilizar todo lo visto en el curso 
en base a las consignas del proyect final.
si bien hay algunas cosas como el carrusel que no lo pude centrar correctamente, utilice
una libreria y el mismo se escuentra adjunto en el proyecto para que funcione.

El proyecto si bien es sencillo esta bastante completo teniendo en cuenta los puntos de la consigna.

Todo el curso de BA me parecio excelente, y muy completo ademas. Seguro se lo recomendaria a alguien para 
que pueda lo realizar.
